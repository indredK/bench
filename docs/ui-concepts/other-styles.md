# Bench 功能 UI 组织 · 其余风格文字说明

> 以下 8 种信息架构风格仅保留文字描述（无 HTML 原型），供对比参考。
> 保留为可交互原型的为：上方标签、下方标签、经典侧边。

---

## 1. 命令面板式（Spotlight / Raycast 风格）
- **核心思路**：⌘K 唤起一个浮动的全局命令面板，所有功能聚合到搜索结果中，按领域分组展示；键盘上下选择、回车直达，鼠标亦可点选。
- **导航逻辑**：无独立导航结构，一切以「搜索 + 快捷命令」为中心；高频功能可置顶/固定。
- **视觉**：深色霓虹质感，半透明磨砂浮层，强调输入光标与选中行的辉光。
- **适合人群**：键盘党、极客、追求「零鼠标」效率的用户。
- **包含能力**：快速启动、端口管理、垃圾清理、环境检测、Token 计算、开发工具、网络诊断、系统信息、系统设置、硬件查询、应用管理、账号管理、术语库，全部可被搜索命中。

## 2. 仪表盘卡片格栅（Dashboard Grid）
- **核心思路**：首页把所有功能铺成卡片网格，按领域分组（开发 / 系统 / 账号 / 应用），每张卡片附带实时状态摘要（如「可清理 3.2GB」「12 个应用待更新」）。
- **导航逻辑**：平铺总览，点卡片进入对应模块；以「一屏看全」为设计目标。
- **视觉**：浅色清爽风、圆角大卡片、状态用色块/徽标提示。
- **适合人群**：新手、总览型用户，希望一眼掌握系统状态。
- **包含能力**：同全部 13 项功能，卡片上直接暴露关键状态与快捷操作。

## 3. 空间画布控制中心（Spatial Canvas）
- **核心思路**：一块可自由拖拽的画布，上面悬浮着各个功能控件面板（小部件），用户自行摆放成个人工作台；面板实时显示状态（端口占用、清理空间、CPU 等）。
- **导航逻辑**：无层级，靠空间位置与个人布局组织；支持面板吸附、缩放、隐藏。
- **视觉**：玻璃拟态 + 渐变背景，强交互、可玩性高。
- **适合人群**：个性化玩家、喜欢 DIY 工作区的用户。
- **包含能力**：常驻型功能为主（端口、清理、系统信息、Token、快速启动），可任选型放置。

## 4. 折叠手风琴（Accordion）
- **核心思路**：单列可折叠分组，全部功能收在「一页」内，按领域折叠展开；顶部搜索框即筛即收。
- **导航逻辑**：单页纵深浏览，展开/收起代替页面跳转，避免上下文丢失。
- **视觉**：浅色平静风，分组标题 + 平滑展开动画，密度可控。
- **适合人群**：浏览型用户、怕被多页面干扰的人。
- **包含能力**：全部 13 项，按「启动 / 工具箱 / 系统 / 资料」四组折叠。

## 5. 双栏主从（Master–Detail）
- **核心思路**：左侧是可搜索的全量模块列表，右侧实时展示选中模块的详情预览（描述、子功能、快捷操作），像一本「功能字典」。
- **导航逻辑**：左列选、右栏读，选中即切换详情，无需进入独立页面。
- **视觉**：中性专业风，左列表 + 右只读预览，强调快速查阅。
- **适合人群**：查阅型用户，想快速了解「每个功能是什么、能干什么」。
- **包含能力**：全部 13 项，右侧展示子功能清单（如系统设置四页、端口管理五能力）。

## 6. 开始菜单磁贴（Start Tiles）
- **核心思路**：左侧窄导航栏 + 右侧分组实时磁贴网格，部分磁贴为宽版或带状态（如「应用管理 12」「账号 5」）。
- **导航逻辑**：左栏切大类、右栏磁贴直达；磁贴可动态反映状态。
- **视觉**：彩色明亮风，磁贴化布局，贴近 Windows 开始菜单心智。
- **适合人群**：习惯 Windows 开始菜单/磁贴交互的用户。
- **包含能力**：全部 13 项，磁贴尺寸可区分主次功能。

## 7. 径向启动器（Radial Launcher）
- **核心思路**：中心是一个枢纽按钮，点击后工具沿环形放射展开成扇区；悬停高亮、指向即选、点击启动，强交互仪式感。
- **导航逻辑**：以「中心 → 环向辐射」组织，无传统列表；适合功能数量中等、追求启动快感。
- **视觉**：深色聚焦背景，放射线与节点动画，中心辉光。
- **适合人群**：启动偏好者、喜欢非常规交互的用户。
- **包含能力**：按领域分环（常用 / 工具箱 / 系统 / 资料），每环挂若干工具。

## 8. 工作区切换器（Workspace Switcher）
- **核心思路**：按人格/场景划分工作区（如「开发」「运维」「日常」「自定义」），切换即换整套工具集与排布；自定义工作区可勾选要显示的功能。
- **导航逻辑**：先选场景、再呈现对应工具，角色化组织而非功能罗列。
- **视觉**：场景标签 + 场景专属配色，切换有过渡。
- **适合人群**：多角色用户（既写代码又管系统还做日常维护）。
- **包含能力**：13 项功能按场景归类，自定义工作区可任意勾选组合。
